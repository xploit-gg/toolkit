Mobile Security Testing

Comprehensive cheatsheet for Android application penetration testing.

⚙️ Environment Setup

Android Setup — Burp Proxy Android 
# Set proxy on device/emulator via ADB
adb shell settings put global http_proxy 192.168.1.100:8080

# Remove proxy
adb shell settings put global http_proxy :0

# Install Burp CA cert
adb push cacert.der /sdcard/
# Settings → Security → Install from storage

# For Android 7+ (Nougat and above) — need network_security_config.xml
# Or use apktool to patch the APK

📱 ADB Commands

Essential ADB 
# List connected devices
adb devices

# Open shell on device
adb shell

# Install APK
adb install app.apk

# Pull file from device
adb pull /data/data/com.target.app/files/config.db ./

# Push file to device
adb push burp.cer /sdcard/

# View logs (filter by app)
adb logcat | grep "com.target.app"

# List installed packages
adb shell pm list packages | grep target

# Get APK path
adb shell pm path com.target.app

# Extract APK from device
adb pull /data/app/com.target.app-1/base.apk ./target.apk

🔬 Static Analysis

APK Decompile with apktool 
# Decompile APK
apktool d target.apk -o output_dir

# Recompile after modification
apktool b output_dir -o patched.apk

# Sign the recompiled APK
keytool -genkey -v -keystore test.keystore -alias test -keyalg RSA -keysize 2048 -validity 10000
jarsigner -verbose -sigalg SHA1withRSA -digestalg SHA1 -keystore test.keystore patched.apk test
zipalign -v 4 patched.apk final.apk
Searching for Secrets in Decompiled Code 
# Find hardcoded API keys / secrets
grep -r "api_key\|secret\|password\|token\|AWS\|Bearer" output_dir/

# Find URLs / endpoints
grep -r "http\|https" output_dir/smali/ | grep -v ".class"

# Check AndroidManifest.xml for permissions and exported components
cat output_dir/AndroidManifest.xml

📌 SSL Pinning Bypass

Bypass with objection 
# Install objection
pip install objection

# Patch APK to include Frida gadget (no root needed)
objection patchapk --source target.apk

# Run patched APK, then connect
objection -g com.target.app explore

# Bypass SSL pinning inside objection shell
android sslpinning disable
Bypass with Frida (rooted device) 
# Start frida-server on device first
# https://github.com/frida/frida/releases → download frida-server for android
adb push frida-server /data/local/tmp/
adb shell chmod 755 /data/local/tmp/frida-server
adb shell /data/local/tmp/frida-server &

# Run SSL bypass script
frida -U -f com.target.app -l ssl_bypass.js --no-pause

# Universal SSL bypass script (save as ssl_bypass.js)
# https://codeshare.frida.re/@pcipolloni/universal-android-ssl-pinning-bypass-with-frida/

Bypass Root Detection with objection

💉 Frida Scripts

List & Hook Methods 
# List loaded classes
frida -U -f com.target.app --no-pause -e "Java.perform(function(){Java.enumerateLoadedClasses({onMatch:function(n){console.log(n)},onComplete:function(){}})})"

# Hook a method — intercept & log arguments + return value
Java.perform(function() {
  var TargetClass = Java.use('com.target.app.LoginActivity');
  TargetClass.checkPassword.implementation = function(pass) {
    console.log('[*] checkPassword called with: ' + pass);
    var result = this.checkPassword(pass);
    console.log('[*] Result: ' + result);
    return result;
  };
});
